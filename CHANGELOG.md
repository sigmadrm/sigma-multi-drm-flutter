## 1.1.0

- Adds Chewie UI controls for a more robust and customizable video player experience.
- Upgrades `video_player_android` dependency to improve performance and stability.
- Fixes a bug where exiting fullscreen mode caused a black screen on Android.
- Refines README documentation and simplifies the example application usage instructions.
- Improves platform support documentation for Mobile, Tablet, and TV.

## 1.0.0

- Initial release of the Sigma Multi-DRM Flutter Plugin.
- Full support for Widevine DRM-protected video streams.
- Support for DASH (.mpd) and HLS (.m3u8) playback.
- Integrated customizable video player controls.
- Implemented secure surface handling for DRM content on Android.
